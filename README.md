# ERC20 Pipe

Multi-chain ERC20 transfer indexer built with [Subsquid Pipes SDK](https://beta.docs.sqd.dev/en/sdk/pipes-sdk/evm.autogenerated/quickstart) and ClickHouse.

## Quick Start

```bash
# Install dependencies
bun install

# Start ClickHouse
bun up

# Index ERC20 transfers
bun eth
```

## What It Indexes

Tracks ERC20 Transfer events and stores:
- Transfer ID (unique across chains)
- Chain ID and block number
- Token contract address
- From/to addresses and amount

Data is stored in ClickHouse optimized for multi-chain queries.

## Configuration

### Add New Networks

1. Create network config in `src/utils/constants/networkConfigs/`
2. Add to `networksConfigs` in `src/utils/constants/network.constant.ts`
3. Add script in `package.json`: `"bnb": "bun src/main.ts bnb"`

### Add Tokens to Track

Edit your network config file:
```typescript
tokens: [
  {
    address: normalizeAddress('0x...'),
    range: { from: START_BLOCK },
  },
]
```

### Add Custom ABIs

```bash
# Place ABI JSON in ./abi/ folder
echo '[...]' > ./abi/token.json

# Generate TypeScript types
bun typegen

# Use in src/main.ts
import * as tokenAbi from './abi/token'
```

## Scripts

- `bun eth` - Index Ethereum mainnet
- `bun all` - Index all configured networks
- `bun up` - Start ClickHouse
- `bun down` - Stop ClickHouse
- `bun typegen` - Generate TypeScript types from ABIs

## Environment

Copy `.env.example` to `.env` and configure:
```bash
CLICKHOUSE_URL=clickhouse://default:password@localhost:8123
LOG_LEVEL=info
```
